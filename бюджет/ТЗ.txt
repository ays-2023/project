Практическое задание: расчет бюджета
Задача
Создать веб-приложение для расчета дневного бюджета с учетом обязательных и необязательных расходов, дополнительных доходов и накоплений.
________________________________________
1. Создание HTML-страницы
В HTML создаем разметку с полями ввода для данных о расходах, доходах и накоплениях.
Добавляем кнопки:
"Начать расчет"
"Утвердить" (для расходов)
"Рассчитать" (для дневного бюджета).
________________________________________
2. Получение данных от пользователя
При старте (по нажатию на кнопку "Начать расчет") спрашиваем:
Ваш бюджет на месяц?
Введите дату в формате YYYY-MM-DD
Эти данные сохраняем в переменные money (бюджет) и time (дата).
________________________________________
3. Создание объекта appData
let appData = {
  budget: money, // бюджет
  timeData: time, // дата
  expenses: {}, // обязательные расходы
  optionalExpenses: {}, // необязательные расходы
  income: [], // дополнительные доходы
  savings: false, // накопления
};

4. Ввод обязательных расходов
Спрашиваем у пользователя 2 раза:
Введите обязательную статью расходов
Во сколько обойдется?
Записываем в объект expenses:
expenses: {
  "Название расхода": "Сумма"
}
5. Расчет дневного бюджета
Вычисляем дневной бюджет:
 

Выводим результат на экран.


6. Расчет уровня достатка
Добавляем функцию detectLevel():
Если бюджет на день < 100 — низкий уровень достатка.
Если 100 – 2000 — средний уровень.
Если > 2000 — высокий уровень.
________________________________________
7. Ввод необязательных расходов
Спрашиваем 3 раза: "Статья необязательных расходов?"
Записываем в optionalExpenses:
optionalExpenses: {
  1: "Название расхода"
}

8. Ввод дополнительных доходов
Пользователь вводит способы доп. дохода (через запятую).
Проверка ввода:
Только строка.
Не может быть пустым.
Нельзя отменить.
Выводим список доп. доходов через forEach, нумерация с 1.
________________________________________
9. Вывод объекта appData в консоль
Используем for in:
for (let key in appData) {
  console.log(`Наша программа включает: ${key} - ${appData[key]}`);
}


10. Работа с HTML
Получаем элементы:
Кнопка "Начать расчет" (id="start")
Все div с class="название-value"
Поля обязательных расходов (class="expenses-item")
Кнопки "Утвердить" и "Рассчитать" (по tagName)
Поля необязательных расходов (class="optionalexpenses-item")
Поля: доходы, чекбокс, сумма, процент, год, месяц, день (querySelector).

11. Итоговый расчет бюджета
Вычитаем обязательные расходы из бюджета.
Рассчитываем чистый дневной бюджет.
Выводим в правую часть экрана.
